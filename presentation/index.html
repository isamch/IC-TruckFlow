IC-TruckFlow

Système de Gestion de Flotte

Application web full-stack pour la gestion intelligente de flottes de camions

Architecture moderne : Backend Node.js + Frontend React

Deux rôles utilisateurs : Administrateurs et Chauffeurs

Gestion complète : véhicules, trajets, maintenance, carburant

Déploiement containerisé avec Docker

Backend - API RESTful Robuste

Stack Technique

Runtime

Node.js avec Express.js v5

Base de données

MongoDB avec Mongoose ODM

Authentification

JWT (Access + Refresh Tokens)

Sécurité

Helmet, CORS, bcryptjs

Fonctionnalités Backend

Gestion des utilisateurs (Admin/Chauffeur)

CRUD complet pour camions, remorques, pneus

Système de trajets avec statuts

Alertes de maintenance automatiques

Tests unitaires avec Jest

Frontend - Interface Utilisateur Moderne

Stack Technique

Framework

React 19 avec Vite

Routing

React Router DOM v7

Styling

TailwindCSS

HTTP Client

Fetch API avec interceptors

Interfaces Utilisateur

Admin

Dashboard, gestion utilisateurs, camions, remorques, pneus, trajets, alertes

Chauffeur

Dashboard personnel, mes trajets, démarrer/terminer trajet, alertes camion actuel

Layout responsive avec sidebar et navigation intuitive

Modules Principaux du Système

Gestion de Flotte

  • Suivi en temps réel des camions
  • Gestion des remorques et pneus
  • Kilométrage et statuts actualisés

Gestion des Trajets

  • Planification et assignation
  • Démarrage/fin de trajet
  • Calcul automatique des distances

Maintenance Intelligente

  • Règles de maintenance personnalisables
  • Alertes automatiques par priorité
  • Blocage de départ si alertes critiques

Logs et Rapports

  • Enregistrement du carburant consommé
  • Historique maintenance par véhicule
  • Génération de rapports PDF

DevOps et Outils de Développement

Containerisation Docker

Docker Compose orchestrant 3 services:

  • MongoDB (base de données)
  • Backend API (port 5000)
  • Frontend React (port 5173)

Volumes persistants pour les données

Network bridge pour communication inter-services

Outils de Développement

Scripts CLI personnalisés pour générateurs

Seeding Database avec Faker.js

Collection Postman pour documentation API

Tests automatisés avec Jest

Sécurité

Variables d'environnement (.env)

HMAC pour codes de vérification
